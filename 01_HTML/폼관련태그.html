폼관련 태그

from 태그는 html에서 사용자가 입력할수 있는 양식을 제공하는 태그이다
from태그 내의 input 태그들을 통해 사용자가 입력한 정보를 서버로 넘기는 역할을 한다
* action속성 : form에 입력된 값들을 전송받을 서버의 클래스명입력
* method 속성 : get/post 방식으로 전송 방식 지정
- get방식 : url 즈소에 form 태그를 통해 넘어가는 데이터가 보임
- post 방식 : url 주소에 form 태그를 통해 넘어가는 데이터 보이지 않음

검색할내용 :

submit 버튼을 눌렀을때 action 지정한 경로로 method에서 지정한 방식에 따라 input 태그에 입력한 값들을 전달한다.
